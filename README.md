kxfw
====
基于Qt的GraphicsView而写的XML-UI界面架构。
<br>
实现的功能包括如下：<br>
1.基本的UI组件，如Label,ComboBox,ProgressBar等常见组件。<br>
2.增加了基于MVC模式的高扩展性的List和Tree控件，已经通过2万多条的记录测试。<br>
3.所有组件都支持XML描述，并依XML控件树结构，生成各种各样的UI界面。<br>
<br>
4.可参考kxfw-example的例子开发产品。<br>
<br>
<br>
演示效果，可到<a href="http://www.kxtry.com">http://www.kxtry.com<a>主页上看。<br>
